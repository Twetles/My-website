HTML Cheat Sheet

The HTML Cheat Sheet page helps you keep track of commonly used elements, tags, and attributes in HTML. Below are some key components you can use in your projects:

  • <html> - Root element of an HTML page
  • <head> - Contains metadata and links
  • <title> - Sets the page title
  • <body> - Contains visible page content
  • <h1> to <h6> - Heading tags
  • <p> - Paragraphs
  • <a> - Hyperlinks
  • <img> - Images
  • <ul>, <ol>, <li> - Lists
  • <div> and <span> - Layout containers
  • <table>, <tr>, <td> - Tables
  • <form>, <input>, <button> - Forms
  • Attributes: id, class, src, href, etc.
  • <meta> - Defines metadata
  • <footer> and <header> - Semantic sections
<!DOCTYPE html>
<html>
  <head>
    <title>My Cheat Sheet</title>
  </head>
  <body>
    <h1>HTML Elements</h1>
  </body>
</html>

CSS Cheat Sheet

The CSS Cheat Sheet page allows you to document key properties, selectors, and styling rules. Below are some key CSS functionalities to remember:

  • Selectors: class, id, element
  • Properties: color, font-size, margin, padding
  • Box Model: width, height, border
  • Display: block, inline, flex
  • Flexbox properties: justify-content, align-items
  • Background properties: background-color, background-image
  • Text properties: text-align, line-height
  • Positioning: absolute, relative, fixed
  • Pseudo-selectors: :hover, :nth-child()
  • Media Queries for responsiveness
  • Gradients: linear, radial
  • Shadows: box-shadow, text-shadow
  • Transitions and Animations
  • Units: px, em, rem, %, vw
  • Colors: HEX, RGB, HSL
body {
  background-color: #f4f4f4;
  color: #333;
  font-family: Arial, sans-serif;
}
h1 {
  color: #6a11cb;
}
